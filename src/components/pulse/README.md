# Pulse Components

This directory contains components for displaying and managing trading pulses.

## Components

### TradeHistory
A table view of trades with features like pagination, filtering, and detailed trade view.

### TradeCalendar
A calendar view that displays trades organized by date, allowing users to:
- See trades aggregated by day
- View daily profit/loss at a glance
- Navigate between months
- Click on days with trades to view details

### PulseChart
Charts trading performance over time with various time range options.

### PulseHeader
Displays pulse information and provides actions like archive, update, and delete.

### PulseStats
Shows statistical information about trading performance.

## Usage

The pulse detail page now includes a toggle between table and calendar views.
- Table view (default): Traditional list format with all trade details
- Calendar view: Visualize trades by date with color-coded results

### Benefits of Calendar View
- Quickly identify trading patterns by day/week
- Spot consecutive winning or losing days
- Visualize trading frequency and consistency
- Gain insights into trading behavior based on calendar patterns
